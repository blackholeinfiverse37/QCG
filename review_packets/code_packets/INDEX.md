# Code Packets Index

- `integration_harness.py`: TANTRA pipeline harness handling network interfaces and lifecycle logic.
- `integration_interfaces.py`: Core abstractions and typed interfaces for all ecosystem capabilities.
- `ecosystem_participation.py`: Simulator configuring local ecosystem participant identities.
- `keshav_live_client.py`: Production HTTP client implementing live KESHAV integration.
- `pritesh_live_client.py`: Production HTTP client implemented to simulate and test live Pritesh quantum integration payloads.
- `web_server.py`: FastAPI server serving as the QCG entrypoint; now includes dynamic capability payload adaptation for Pritesh's quantum payloads.
- `live_integration_evidence.py`: Automated harness verifying and collecting evidence for all available ecosystem endpoints.
